# Recipe Sharing Network

A community recipe sharing application for discovering, posting, and saving favorite recipes.

## Features

- Create, edit, and share recipes
- Browse recipes by category or ingredient
- Save favorites for later
- User-friendly interface for home cooks and food lovers

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Change into the project directory:
   ```bash
   cd recipe-sharing-network
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Start the development server:

```bash
npm start
```

Open the app in your browser at `http://localhost:3000`.
